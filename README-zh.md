# window-helper

[English](./README.md) | 简体中文

## 描述

这是一个用于隐藏窗口的程序。

仅限于 Windows。

## 如何使用

需要 g++

首先，克隆这个仓库
```bash
git clone https://github.com/EarthMessenger/window-helper.git
```

并编译运行
```bash
cd window-helper
g++ ./main.cpp -o window-helper
./window-helper
```

使用 `<shift>+R` 获取窗口句柄。

使用 `<shift>+Z` 隐藏/显示窗口。
